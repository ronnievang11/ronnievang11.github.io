UI vs UX

It is very fascinating to understand the difference between a UI/UX designer and web designer. Upon watching this video, I am able to see the contrast of web pages. I really like the way how Mike Locke describe the web pages that you are a web designer if the pages are like a brochure where users are reading and viewing for information and that you are a UI/UX designer if there are some interaction between the web page and the user. to be a UI/UX designer, you have to understand what both term does first. A UX Design refers to the term User Experience Design and a UI Design is User Interface Design. Both of them work closely together to a product, but in vary ways. UX Design is more to the technical side where it provides customer satisfaction and loyalty by improving the usability, ease of use, and pleasure provided by the interaction between the customer and the product. While UI Designer present the look and feel interactivity of a product. It deals more to the graphic design side of the web page. The video shows a great process of how UI/UX design work and being created, gives you that being a UI/UX designer is a fun job. Like Mike Locke says, you don’t have to code all of these, you just need to know the basic of a web designer, Photoshop and Fireworks, and understand flowchart or workflow.

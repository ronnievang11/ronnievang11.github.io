Website vs App

There is a broader clarification about the difference between a website and a web application. It made me realize that there were more depth into a website and a web application. The video explains how websites are informational websites where they provide information with a description and contact. Opposing to web application, it provides information, but can influence the visitors in wanting to do something with the provided information. It surprises me that such good resource like the library website can be a website application. Another thing that the video provided was about how to create a proper business website. He talks about if creating a website for a business, it must be responsible to safeguard with the information such as personal information login and payments that it obtains from customers, visitors, and users. When creating a website for a business, the business has to decide if they want a website or website application. As well, a unique idea from the video was website web application hybrid; making the website as an informative page, but then later including application. He also ended off in the video with a great way of saying that it is always easier and less expensive to hire right web development agency that can grow with your startup and meeting your development needs from day. An example of a website provided with information is the CIA. website. Also, one example for a website application can be gofundme, where this site is use for fundraising.
